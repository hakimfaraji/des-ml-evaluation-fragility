# Zenodo metadata draft

Title: des-ml-evaluation-fragility: Reproducibility package for leakage-clean DES recommendation evaluation

Version: v1.1

Description:
This record contains the curated dataset, code, audit outputs, evaluation outputs, feature-importance results, paired statistical-comparison outputs, and revised figures associated with the manuscript "When High Accuracy Misleads in Literature-Derived Machine Learning for Deep Eutectic Solvent Recommendation".

The package supports reproduction of the leakage-clean DOI-grouped evaluation, train-only preprocessing, train-only pairwise construction, test-only within-group inference, sanity-baseline evaluation, permutation feature importance, paired ranking effect sizes, and uncertainty/statistical comparisons reported in the revised manuscript and Supporting Information.
