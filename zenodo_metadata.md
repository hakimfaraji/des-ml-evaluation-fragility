## Description

This repository contains the data, code, audit outputs, and reproducibility resources associated with the manuscript "When High Accuracy Misleads in Machine Learning for Deep Eutectic Solvent Recommendation".

The study evaluates the reliability of machine-learning-based recommendation for deep eutectic solvent systems under literature-derived experimental conditions. It provides a leakage-clean computational workflow, a curated experiment-level dataset, and the principal audit and evaluation outputs required to reproduce the analyses reported in the manuscript.

The repository includes:

- Dataset S1: curated experiment-level dataset for DES-based pesticide extraction;
- Code S1: reproducible leakage-clean computational workflow;
- audit outputs documenting split integrity, feature filtering, and train/test isolation;
- pairwise dataset summaries;
- ranking and classification metrics;
- uncertainty and statistical evaluation outputs;
- sanity baseline results.

## Notes

This archive is intended to support reproducibility of the leakage-clean machine-learning workflow and the principal results reported in the associated manuscript. The repository does not claim to provide a general-purpose DES recommendation engine; it provides the data and code required to reproduce the analyses described in the study.
