# Zenodo v1.2 release checklist

This release is an author-metadata correction and final reproducibility package update.

## Before publishing

1. Confirm that the current manuscript and Supporting Information use the same author list as this package metadata.
2. Confirm that `CITATION.cff` and `zenodo_metadata.md` do not include authors removed from the revised manuscript.
3. Upload the complete repository contents to GitHub.
4. Create a GitHub release tagged `v1.2`.
5. Let Zenodo archive the release and assign the new DOI.
6. Replace the previous DOI in the manuscript, SI, and Data Availability Statement with the DOI for the v1.2 release.

## Suggested GitHub release title

`v1.2 - Author-metadata corrected reproducibility package`

## Suggested release note

This release updates the repository metadata to match the current revised manuscript author list and retains the revised reproducibility package, including Dataset S1, Code S1, audit outputs, ranking and classification outputs, sanity-baseline results, feature-importance outputs, paired effect-size outputs, revised tables, and revised figures.
